# hw3_yolov5 - Детекция беспорядка на диване (YOLOv5)

## Описание
Обучение модели YOLOv5 для детекции двух состояний дивана: **порядок** (`clear`) и **беспорядок** (`mess`).  
Датасет собран вручную (134 фото), размечен в Roboflow. Проведено 3 эксперимента с разными версиями датасета и весами классов.

Лучшая модель (`exp1_50e_v2`) достигла mAP50 = 0.995 на валидации и уверенно детектит беспорядок на видео (0.90-0.95). Эта модель выбрана, потому что она оказалась наиболее стабильной на видео с порядком — остальные модели чаще ошибались.
