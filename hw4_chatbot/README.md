## Описание проекта

Разработан **intent‑based чат‑бот** на PyTorch, который помогает ребёнку получить пароль от компьютера.  
Бот проверяет выполнение заданий, собирает оценки и назначает дополнительные задания в зависимости от среднего балла.

**Сценарий работы:**
1. **Главные задания** (поесть, сложить одежду, помыть посуду, сделать уроки)
2. **Выбор предметов**, которые были в школе
3. **Ввод оценок** по каждому предмету (можно ввести `нет`, если оценки не было)
4. **Расчёт среднего балла** и назначение дополнительных заданий
5. **Выдача пароля** после выполнения всех заданий

Бот работает в **консоли** и **Telegram** (с инлайн‑кнопками).  
Все диалоги сохраняются в лог‑файлы для проверки.

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/DziyanaLysak/BelHard_DS_2.git
cd BelHard_DS_2/hw4_chatbot
```

### 2. Создать виртуальное окружение и активировать
```bash
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
.venv\Scripts\activate         # Windows
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Скачать данные NLTK
```bash
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab')"
```

### 5. Обучение модели
```bash
python src/train.py
```

После обучения модель сохранится в `models/` и как `data.pth` в корне.

### 6. Запуск консольной версии
```bash
python src/chat_console.py
```

### 7. Запуск Telegram‑бота
1. Создайте файл `.env` в корне `hw4_chatbot/` и добавьте токен:
   ```
   BOT_TOKEN=ваш_токен_от_BotFather
   ```
2. Запустите бота:
   ```bash
   python telegram_bot/bot.py
   ```

## Эксперименты с гиперпараметрами

Проведены 3 эксперимента. Лучшая конфигурация:

| Параметр | Значение |
