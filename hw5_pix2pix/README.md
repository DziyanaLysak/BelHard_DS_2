# hw5_pix2pix - Pix2Pix (день → ночь)

## Описание

Проект реализует генеративно-состязательную сеть **Pix2Pix** для перевода дневных изображений в ночные на датасете `night2day`.

- **Генератор**: U-Net с skip-connections
- **Дискриминатор**: PatchGAN (70×70)
- **Обучение**: 100 эпох, batch_size=4, оптимизатор Adam (lr=0.0002), λ=100

## Результаты

- Наша модель (100 эпох) лучше передаёт детали на городских сценах.
- Официальная модель (15 эпох) даёт более сглаженную картинку на природных ландшафтах.
- Увеличение эпох улучшает детализацию ценой появления пикселизации.

Модель обучена и протестирована. Pix2Pix способен решать задачу перевода "день → ночь", но требует тонкой настройки гиперпараметров и достаточного количества эпох для достижения высокого качества.
